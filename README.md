# Simple Triangulation 2D
A simple Python script to demonstrate the subdivision of a planar object into triangles.

## Requirements

 - [Python 3](https://www.python.org/)
 - [NumPy](http://www.numpy.org/)

 
## Run

**Initialize**

    pyvenv env

**Start**

    source env/bin/activate

**Stop**

    deactivate

```
./triangulation.py
```
